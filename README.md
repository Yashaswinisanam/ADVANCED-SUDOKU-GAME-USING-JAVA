# ADVANCED-SUDOKU-GAME-USING-JAVA
An interactive Sudoku game developed using Java Swing with multiple difficulty levels, real-time validation, hint system, timer functionality, and solution checking.
## Features
- Dynamic Sudoku puzzle generation
- Easy, Medium, and Hard difficulty levels
- Real-time input validation
- Hint system for assistance
- Solution checking and display
- Timer tracking gameplay duration
- Interactive Java Swing GUI
- Exit confirmation dialog
## Tech Stack
- Java
- Java Swing
- Object-Oriented Programming (OOP)
- Event Handling
- Backtracking Algorithm
## How to Run

1. Clone the repository
2. Open project in IntelliJ IDEA / Eclipse
3. Compile and run the main Java file
4. Start playing Sudoku
## Project Structure

- GUI Components
- Puzzle Generator
- Validation Logic
- Timer Module
- Hint & Solution System
